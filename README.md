# Benchtool - julia

Simple benchmark, using:

  * Julialang ( Developed and tested on 1.2 )


### Example usage
For the loop bench
```
julia src/main.jl 1 99999
```

### Params:
 1. the bench type 1 = loop, 2 = recursive
 2. the fibonacci ***n*** number to calculate
 3. the exection count ( default 5 )
